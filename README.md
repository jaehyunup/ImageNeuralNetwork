Dive-in Background Subtraction
=============
This project is the process of developing a program that learns the background area of ​​the first frame of the video and guesses and deletes the background from the video.


1.개발 목적
--------------------------------
영상에서의 BackgroundSubtraction 에 대한 이해도를 높히기 위해 개발중인 개인 프로젝트


2.개발 환경
-------------
* Python-3.6.8 
* OpenCV-3.4.5-contrib 
* numpy
* tensorflow
* matplotlib
* keras
